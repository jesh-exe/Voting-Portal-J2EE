# It is a Dynamic Java Project 
## VOTING PORTAL
1. Servlets
2. Session Management
3. Init Params
4. JDBC
5. Client Pull (Redirection)

### Functionalities
1. Email Check
2. Age Check
3. Vote Repetition Check
4. Invalid Login Check
5. Sign Up
6. Add New Candidates
7. Admin Dashboard - For Admin
8. User Dashboard - For Users
